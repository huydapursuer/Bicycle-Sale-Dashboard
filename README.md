# Bicycle-Sale-Dashboard


We analyzed the Bicycle Sales Dataset (https://www.kaggle.com/datasets/jayavarman/bike-sales-data-of-100k) and built a dashboard with three sections: Marketing Insight  (orders and customer trends), Finance Insight (revenue and profit analysis), and Solutions & Recommendations

Access the BI Dashboard via: https://app.powerbi.com/groups/me/reports/a0f0f2bd-22fc-4328-9e08-7673a9f29480/ReportSection34284191d702d7a6ad07?experience=power-bi
